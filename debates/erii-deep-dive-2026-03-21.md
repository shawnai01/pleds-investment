## Phase 3: Value Chain (L3) — 담수화 밸류체인

### 밸류체인 매핑
```
원수 취수 → 전처리 → 고압펌프 → RO 멤브레인 → 에너지회수(PX) → 후처리 → 배수
                         ↑                    ↑              ↑
                      Grundfos/            DuPont/          ★ ERII ★
                      Flowserve           Toray/LG          (PX 독점)
```

### 🏗️ The Strategist (Porter/Dorsey)
[THESIS] ERII의 PX Pressure Exchanger는 담수화 밸류체인에서 가장 좁은 병목 중 하나:
- **전환 비용:** RO 플랜트 설계 단계에서 PX를 중심으로 시스템 최적화. 교체 시 전체 재설계 필요.
- **규모 경제:** 230명으로 $135M 매출 = 인당 $587K 매출. 매우 자산경량(asset-light).
- **네트워크:** 설치 기반(installed base)이 aftermarket 매출 생성. 누적 설치 대수가 경쟁 진입 장벽.

### 🔗 Network Thinker
[THESIS] 네트워크 효과는 제한적이나 **설치 기반 효과(installed base effect)**가 존재. 전 세계 대형 담수화 플랜트의 대부분에 PX가 설치되어 있다면, 신규 플랜트도 실적 검증된 PX를 선택할 유인 강함. 이는 "de facto standard" 효과.

## Phase 5: Cross-Layer — Bottleneck Hunter

### PX 독점이 진짜 병목인가?

**병목 분석:**
1. [THESIS] 담수화 RO 공정에서 에너지가 총 비용의 40-50%를 차지 [미확인 — 산업 보고서 참조 필요]. PX는 이 에너지의 최대 60%를 회수. 따라서 PX 없이는 담수화 경제성이 크게 약화.
2. [THESIS] PX는 "있으면 좋은 것"이 아니라 **필수 구성품**. 대형 RO 플랜트에서 PX 없이 경쟁력 있는 담수 생산 비용 달성 불가능.
3. [THESIS] 이는 ERII를 담수화 밸류체인의 **병목 소유자(Constraint Owner)**로 위치시킴.

### 15년간 경쟁자 진입 실패 이유

[THESIS] 복합적 요인:
1. **기술 장벽:** PX는 정밀 세라믹 가공 + 유체역학 노하우가 결합된 기술. 특허뿐 아니라 제조 know-how가 핵심.
2. **시장 규모:** 전체 ERD 시장이 $200-300M 수준 [미확인] — 대기업이 진입하기엔 너무 작고, 스타트업이 진입하기엔 기술 장벽이 높은 "골디락스 니치."
3. **설치 기반:** 기존 플랜트 운영자들의 PX 신뢰도 + 실적 데이터 → 신규 진입자의 검증 비용이 매우 높음.
4. **규제/인증:** 대형 담수화 프로젝트는 검증된 장비만 허용. 신규 기술의 승인 기간이 수년.

[THESIS] **핵심 판단:** 이는 시장 규모 + 기술 장벽의 복합 효과. "시장이 커지면 경쟁자도 진입한다"는 가능하나, 10년 이상의 리드 타임이 존재. 단기-중기(3-5년)에는 해자 건재.

### Bottleneck Hunter 최종 판정
- 🔴 **현재 병목:** 담수화 프로젝트 파이프라인 자체 (수요 측). ERII 기술이 아닌 프로젝트 발주 타이밍이 병목.
- 🟡 **차기 병목:** 담수화 확대 시 멤브레인 공급 or 인력 부족이 될 수 있으나, PX는 병목이 아님 (ERII 생산능력 여유 추정 [미확인])
- 🟢 **병목 소유자:** ERII는 에너지회수 기술의 독점 공급자로서 확고한 Constraint Owner.
