         ┌──────────────────────────────────────────────────────────┐

         │  변압기 → 스위치기어 → UPS → PDU → Busway → 랙 PDU → CDU │
         └──────────────────────────────────────────────────────────┘

**[FACT] "Grid-to-Chip" 수직통합의 구체적 의미**

Eaton은 2025년 Investor Day에서 "Grid-to-Chip" 전략을 공식화:
1. **Grid 레벨**: 중압 스위치기어, 변압기, 그리드 연결 장비
2. **Building 레벨**: 저압 배전반, UPS, 전력품질 장비
3. **IT 레벨**: PDU, 랙 마운트 UPS, Busway
4. **Chip 레벨 (Boyd 인수)**: 액체냉각 CDU, 콜드 플레이트

[THESIS] 이 수직통합은 **단일 벤더 솔루션** 제공 → 설계 복잡성 감소, 총소유비용(TCO) 절감 → 고객 Lock-in 강화

## 3. 기업 경쟁력 (Competitive Moat)

### 3.1 해자 분석 (5가지)

**1. 규모의 경제 — 강함 ⭐⭐⭐⭐**

[FACT] Eaton의 글로벌 제조 풋프린트:
- 175개국 운영, 90,000+ 직원
- $27.4B 매출 → R&D, 마케팅, 제조 효율에서 규모 우위
- Electrical Americas만 $13.5B로 Hubbell 전체 매출의 2.5배

**2. 전환 비용 (Switching Cost) — 매우 강함 ⭐⭐⭐⭐⭐**

[THESIS] 전력 인프라의 전환 비용은 극도로 높음:
- **물리적 Lock-in**: 스위치기어, 버스웨이는 건물 구조와 통합 — 교체 시 대규모 공사 필요
- **규격 호환성**: Eaton 장비는 Eaton 부품과 최적 호환 — 혼용 시 성능/안전 저하
- **서비스 계약**: 장기 유지보수 계약 체결 → 관계 장기화
- **인증/승인**: 변경 시 재인증 필요 (UL, IEC 등) — 시간 6-12개월, 비용 수십만 달러

**3. 브랜드/인증 장벽 — 강함 ⭐⭐⭐⭐**

[FACT] 진입 장벽:
- UL, IEC, IEEE 인증 획득: 수년 소요, 수백만 달러 투자
- 데이터센터 고객의 Approved Vendor List 진입: 2-3년 검증 기간
- 유틸리티 공급자 자격: 수십 년의 트랙레코드 필요
- Eaton 브랜드: 1911년 설립, 113년 역사

**4. 기술적 해자 — 중-강 ⭐⭐⭐⭐**

[THESIS] 전력관리는 "로켓 사이언스"는 아니지만, 통합 노하우가 핵심:
- 시스템 설계 전문성: Grid-to-Chip 통합 설계 역량
- 소프트웨어/디지털: Brightlayer 플랫폼 (DCIM, 예측 유지보수)
- 특허 포트폴리오: 10,000+ 특허 보유
- [주의] 개별 제품 기술보다 **시스템 통합 역량**이 진정한 해자

**5. 네트워크 효과 — 약함 ⭐⭐**

[THESIS] 직접적 네트워크 효과는 제한적:
- 고객 간 직접 연결 없음
- 단, **설치 기반 효과** 존재: 더 많은 Eaton 장비 설치 → 더 많은 서비스 엔지니어 양성 → 더 나은 서비스 → 더 많은 고객
- 소프트웨어 플랫폼 확대 시 데이터 기반 개선 가능성

**종합 해자 등급: Wide Moat (넓은 해자)**
