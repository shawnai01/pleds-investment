## 🔴 Phase 1: Bear Thesis 구축

### Bear Thesis 1: 밸류에이션 정당화 불가능 ⚠️⚠️⚠️

**현재 Implied Assumptions (P/E 150 정당화 조건):**

```
현재 시총 $40B ÷ 2026E 순이익 = P/E ~140
→ 이게 정당화되려면:
   1) 향후 5년 평균 30%+ 성장 유지 (현재 27% 가이던스)
   2) SBC가 매출 대비 10% 이하로 감소
   3) EBITDA 마진 28%+ 달성 (현재 25.5%)
   4) 경쟁 위협 제로
   5) AI 해자 유지
```

**Reality Check:**

| 가정 | 현실 | 판정 |
|------|------|------|
| 30%+ 성장 5년 | 가이던스 27-30%, TAM 침투율 4% | ⚠️ 가능하나 점점 어려워짐 |
| SBC 10% 이하 | 22% → 2.5% 희석만 제한, 절대액 유지 | 🔴 불가 |
| EBITDA 28%+ | 25.5% → 28% 목표 (2028) | ⚠️ 시간 필요 |
| 경쟁 제로 | Motorola $4.4B 인수로 공격 | 🔴 위협 증가 |
| AI 해자 | GPT-4 기반, 범용 모델 | 🔴 Commoditized |

**[THESIS]** P/E 140-150은 **"완벽한 실행 + 경쟁 부재 + AI 독점"**을 가정 — 현실과 괴리

### Bear Thesis 2: Draft One AI = Commodity 🔴

**검증된 사실:**
- **[FACT]** Draft One은 **OpenAI GPT-4 Turbo** 기반 (Axon 공식 문서)
- **[FACT]** Azure OpenAI Service 경유 → Microsoft 의존
- **[FACT]** 자체 개발 AI 모델 **없음**

**Commoditization 테스트:**

| 질문 | 답변 |
|------|------|
| AI 모델 독점? | ❌ GPT-4 = 누구나 사용 가능 |
| 데이터 독점? | ⚠️ 바디캠 데이터 있으나 학습에 미사용 |
| Workflow 통합? | ✅ Axon 생태계 통합 |
| 경쟁사 복제 가능? | ✅ Motorola도 CommandCentral + AI 개발 중 |

**[THESIS]** Draft One은 기술적 해자가 **제로**
- 경쟁사가 GPT-4/Claude 연동하면 동일 기능 구현 가능
- 유일한 방어선 = Switching Cost (Evidence.com 데이터 lock-in)
- 하지만 신규 고객은 처음부터 Motorola 선택 가능

### Bear Thesis 3: Motorola의 진짜 위협

**[FACT]** Motorola Solutions, 2025년 **Silvus Technologies $4.4B 인수**
**[FACT]** Motorola V700 바디캠 + CommandCentral 소프트웨어 보유
**[FACT]** LMR(Land Mobile Radio) 시장 **90%+ 독점** → 번들 전략 가능

**경쟁 시나리오:**

```
Motorola 전략:
1. 라디오 교체 시 V700 바디캠 번들 제안
2. CommandCentral에 AI 보고서 기능 추가 (GPT-4/Claude)
3. 대형 도시 계약에서 "All-in-one" 딜 공격

Axon 취약점:
- 라디오 없음 → 번들 불가
- 신규 고객 확보에서 열세 가능
- Evidence.com 전환 비용이 유일한 방어선
```

**[THESIS]** Motorola는 2-3년 내 Axon의 신규 계약 30-40% 가로챌 수 있음
