# Phase 3: Variant Perception (변이 인식)

## 시장 컨센서스

| 항목 | 시장 견해 |
|------|----------|
| **밸류에이션** | 단기 역풍 반영, 저평가 인정 |
| **건설 사이클** | 2026년 둔화 지속, 2027년 회복 |
| **인프라 수혜** | 장기 긍정, 단기 지연 |
| **컨센서스 레이팅** | Hold~Buy 혼재 |
| **목표가** | $22-26 범위 (중앙 $24-25) |

## 🔴 시장이 과소평가 중인 것 (Bear Variant)

1. **가동률 격차의 구조적 의미**
   - 시장은 사이클 탓으로 치부
   - 실제로는 MGRC 대비 **경쟁력 열위** 반영 가능
   - 같은 환경에서 16pp 차이 = 고객 선호도/효율성 문제

2. **M&A 실패의 장기 영향**
   - $180M은 일회성 비용으로 처리됐지만
   - 시장 지배력 강화 기회 영구 상실
   - 경쟁사 MGRC는 독립 생존 + 경쟁력 유지

3. **IIJA 수혜 과대 추정**
   - 시장은 "언젠가" 수혜 기대
   - 실제: 2026년 9월까지 배포 → 착공은 2027-2028
   - **WSC 가이던스에 IIJA 수혜 미반영 = 경영진도 회의적**

## 🟢 시장이 과대평가 중인 것 (Bull Variant)

1. **역성장 가이던스의 보수성**
   - 경영진: "터너라운드 가정 안 함"
   - 주문 잔고 +17%와 괴리 = **보수적 기대치**
   - 상회 가능성 존재

2. **구조조정 비용의 일회성**
   - $302M은 2025년에 인식 완료
   - 2026년은 클린 베이스로 비교 가능
   - 플릿 효율화 효과 발현 가능
