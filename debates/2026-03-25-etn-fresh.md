## Executive Summary

**Eaton Corporation (ETN)**은 전력관리 인프라의 핵심 플레이어로, AI/데이터센터 전력 수요 폭발의 직접 수혜주다. Q4 2025 데이터센터 주문 +200%, 백로그 +31% YoY 기록적 성장을 보이며, Boyd Thermal $9.5B 인수로 Grid-to-Chip 솔루션을 완성했다.

**그러나 Bear-First 관점에서:** 현재 P/E 32x는 10년 평균 대비 44% 프리미엄이며, Boyd Thermal 인수 밸류에이션(22.5x EBITDA)도 공격적이다. 2027년 CapEx 사이클 피크 리스크와 경쟁 심화(Schneider, Vertiv)가 주요 우려점.

| 핵심 지표 | 값 |
|----------|-----|
| 현재가 | $357-360 |
| TTM P/E | 32-34x |
| Forward P/E | 24-27x |
| 2026E EPS | $13.00-$13.50 |
| 백로그 | $19.6B (+29% Electrical) |
| 데이터센터 주문 | +200% (Q4) |
| Boyd Thermal | $9.5B 완료 (3/12) |

## 1. 투자 가설 (THESIS/FACT)

### THESIS (검증 필요)
1. **AI 전력 슈퍼사이클이 2030년까지 지속**되며, ETN이 Grid-to-Chip 수직통합으로 최대 수혜
2. **변압기 공급 병목**(리드타임 143주)이 ETN의 가격 결정력과 마진을 지지
3. **Boyd Thermal 인수**로 액체냉각 시장 선점, 데이터센터 토탈 솔루션 완성
4. **Mobility 분사**로 포트폴리오 단순화, 고성장 Electrical/Aerospace 집중

### FACT (검증됨)
1. ✅ Q4 2025 EPS $2.91 (기록), FY25 마진 24.5% (기록)
2. ✅ Electrical Americas 백로그 +31% YoY → 기록적 수준
3. ✅ 데이터센터 주문 +200%, 매출 +40% (Q4 2025)
4. ✅ Book-to-Bill 1.1x (수주 > 매출)
5. ✅ Boyd Thermal 인수 완료 (2026-03-12, $9.5B)
6. ✅ Mobility 분사 발표 (2026-01-26, 2027 Q1 완료 예정)
7. ✅ 2026 가이던스: 유기적 성장 7-9%, EPS $13.00-$13.50

### Bear Case (의무 검토)

**Bear 1: 밸류에이션 과열**
- TTM P/E 32x는 10년 평균 대비 **44% 프리미엄**
- PEG 3.11 → 성장률 대비 비싼 수준
- VRT P/E 77x보다 낫지만, ABB 22x, Schneider 25x 대비 프리미엄
- **위협 수준: 중간** — 성장 가시성(백로그)이 부분 정당화

**Bear 2: Boyd Thermal 인수 리스크**
- $9.5B, 22.5x EBITDA = **공격적 밸류에이션**
- 액체냉각 시장 경쟁 심화 (Vertiv, Schneider 모두 공격적)
- 통합 실패 시 write-down 리스크
- **위협 수준: 중간-높음** — 인수 통합 1년 모니터링 필수

**Bear 3: 2027년 사이클 피크**
- 하이퍼스케일러 CapEx가 2026년 $600B로 정점 도달 가능
- AI ROI 검증 지연 시 2027년 투자 속도 조절
- ETN 데이터센터 매출 비중 급증 → 의존도 상승
- **위협 수준: 중간** — 다각화(유틸리티, Aerospace)가 완충

**Bear 4: 경쟁 심화**
- Schneider Electric: 글로벌 1위, 유럽 강점
- Vertiv: 퓨어플레이 DC 인프라, P/E 77x라도 성장 집중
- ABB: 전기화/자동화 확장, 가격 경쟁
- **위협 수준: 낮음-중간** — ETN의 북미 제조 강점이 관세 환경에서 유리

## 5. 리스크 대비 상방 (R:R, Bear/Base/Bull+확률, EV)

### 시나리오별 수익률 (현재가 $360 기준)

| 시나리오 | 목표가 | 수익률 | 확률 | 가중 수익 |
|----------|--------|--------|------|-----------|
| **Bull** | $450 | +25% | 20% | +5.0% |
| **Base** | $410 | +14% | 55% | +7.7% |
| **Bear** | $310 | -14% | 25% | -3.5% |
| **Expected Value** | — | — | 100% | **+9.2%** |

### Risk-Reward Ratio (4R Framework)
```
Bull Upside: $450 - $360 = +$90 (+25%)
Bear Downside: $360 - $300 = -$60 (-17%)

R:R Ratio = 90 / 60 = 1.5 : 1

확률 가중 R:R = (0.20 × 90 + 0.55 × 50) / (0.25 × 60) = 45.5 / 15 = 3.0 : 1
```

### Expected Value 분석
- **EV = +9.2%** (12개월)
- 무위험 수익률(5%) 대비 **+4.2% 초과수익** 기대
- **결론**: 양의 기대값, 투자 적합

### Kelly Criterion (참고)
```
Kelly % = (bp - q) / b
b = Bull 수익률 / Bear 손실률 = 25% / 17% = 1.47
p = Bull+Base 확률 = 0.75
q = Bear 확률 = 0.25

Kelly = (1.47 × 0.75 - 0.25) / 1.47 = 0.58 = 58%

Half-Kelly (보수적) = 29%
권장 비중: 10-15% (매우 보수적)
```
