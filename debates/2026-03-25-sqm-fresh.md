## Phase 7: Risk-Reward & Expected Value

### R:R 계산 (진입가 $55 기준)

```
Upside (Base): $70 - $55 = $15 (+27%)
Downside (Bear): $55 - $42 = $13 (-23%)
R:R = 15/13 = 1.15:1

Upside (Bull): $100 - $55 = $45 (+82%)
Weighted Upside = (0.45 × $15) + (0.20 × $45) = $6.75 + $9 = $15.75
Weighted Downside = 0.35 × $13 = $4.55
R:R = 15.75/4.55 = 3.5:1 (확률 가중)
```

### Expected Value (현재가 $77 기준)

```
EV = Σ(확률 × 수익)
Bear: 0.35 × ($45 - $77) = 0.35 × (-$32) = -$11.20
Base: 0.45 × ($70 - $77) = 0.45 × (-$7) = -$3.15
Bull: 0.20 × ($100 - $77) = 0.20 × $23 = +$4.60

EV = -$11.20 - $3.15 + $4.60 = **-$9.75** (12M 기준)
EV per share = **-$9.75** → 현재가 대비 -12.7%
```

**결론**: 현재가($77) 기준 **기대값이 음수**. 진입 부적합.

### Expected Value (진입가 $55 기준)

```
Bear: 0.35 × ($45 - $55) = -$3.50
Base: 0.45 × ($70 - $55) = +$6.75
Bull: 0.20 × ($100 - $55) = +$9.00

EV = -$3.50 + $6.75 + $9.00 = **+$12.25** (12M 기준)
EV per share = **+$12.25** → +22% 기대수익률
```

**결론**: $55 진입 시 기대값 양수. 합리적 진입점.
