## 핵심 요약 (Executive Summary)

VRT는 AI 데이터센터 인프라의 **핵심 병목**(냉각/전력)을 해결하는 사실상의 승자. $15B 백로그(+109% YoY)와 252% 오더 성장은 압도적. 그러나 **P/E 73.5x**는 industry 30.8x의 2.4배로, 완벽한 실행을 이미 가격에 반영. S&P 500 편입 후 추가 상승으로 **리스크 대비 보상이 악화**됨. Bear-First 관점에서 **진입 시점은 아니며, 조정 대기가 합리적**.

## 5. 리스크 대비 상방 (R:R, Bear/Base/Bull+확률, EV)

### 4R 프레임워크 분석

**현재가 $260 기준 진입 시:**

| | Price | Return | 확률 |
|---|-------|--------|------|
| Bear | $165 | -36.5% | 25% |
| Base | $280 | +7.7% | 50% |
| Bull | $405 | +55.8% | 25% |

**Expected Value (EV):**
```
EV = (25% × -36.5%) + (50% × 7.7%) + (25% × 55.8%)
   = -9.1% + 3.9% + 14.0%
   = +8.7%
```

**Risk/Reward Ratio:**
- 상방 (Base to Bull): +$20 ~ +$145
- 하방 (Base to Bear): -$95
- **R:R = 0.9:1** (불리함)

### 🔴 현재 진입은 비합리적

**이유:**
1. EV +8.7%는 무위험 수익률(5%) 대비 불충분한 프리미엄
2. R:R 0.9:1은 최소 기준 2:1 미달
3. 하방 리스크 -36.5%가 상방 +55.8%보다 확률 가중 시 크게 느껴짐

### 권장 진입가 $200에서 재계산

| | Price | Return | 확률 |
|---|-------|--------|------|
| Bear | $165 | -17.5% | 25% |
| Base | $280 | +40% | 50% |
| Bull | $405 | +102.5% | 25% |

**EV @ $200:**
```
EV = (25% × -17.5%) + (50% × 40%) + (25% × 102.5%)
   = -4.4% + 20% + 25.6%
   = +41.2%
```

**R:R @ $200 = 3.3:1** ✅ 훨씬 매력적
